# Vue.JS Work - A2 DW2 2021
## Getting Started
### Prerequisites

To install the project, you need :
* npm

### Installation

Execute `npm install` to install the dependencies.

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
